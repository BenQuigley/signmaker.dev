About signmaker.dev

“signmaker” is an old screen name. One of my favorite characters is the Sign Painter from the puzzle game World of Goo. I call my site signmaker.dev because putting up signs (or labels, web sites, code comments…) is a pro-social, semantically deep act that can have profound consequences in a system. As you might expect, I own a label maker and know how to use it.

On a less philosophical note, this site is made with open-source software:

  • it is built using pelican
  • and a theme forked from Alexander Revicenzi,
  • and is hosted on Netlify.
